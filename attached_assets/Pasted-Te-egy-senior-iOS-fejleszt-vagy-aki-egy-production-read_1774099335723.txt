→ Megoldás: guard let vagy ?? default value

6.6 MAVLink CRC checksum implementálás
swift// ITU X.25 CRC-16 a MAVLinkWrapper.swift buildPacket() metódusban:
private func crc16(data: Data, seed: UInt16 = 0xFFFF) -> UInt16 {
    var crc = seed
    for byte in data {
        var tmp = UInt16(byte) ^ (crc & 0xFF)
        tmp ^= (tmp << 4) & 0xFF
        crc = (crc >> 8) ^ (tmp << 8) ^ (tmp << 3) ^ (tmp >> 4)
    }
    return crc
}

## App Review Notes
This is a motorcycle safety app that uses iOS motion sensors to detect
accidents using the MAIDS standard algorithm. The app requires:

1. Background location: continuously monitors rider position for
   accident detection and ride tracking.

2. Emergency call (112/911): CallKit integration requires user
   voice confirmation before connecting. The call is NEVER made
   automatically – a countdown timer gives the rider 30 seconds
   to cancel. If the rider is unresponsive, only SMS is sent
   automatically.

3. Microphone: used exclusively for on-device speech recognition
   (WhisperKit). No audio is sent to any server.

Demo account for review: No login required.
Test crash detection: Shake phone vigorously for 2 seconds.
