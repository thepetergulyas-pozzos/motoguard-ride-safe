const isDark = colorScheme === 'dark';

useTheme hook (ez legyen az egyetlen téma hook):
javascriptimport { useColorScheme } from 'react-native';
import { Colors } from '../constants/Colors';

export function useTheme() {
  const scheme = useColorScheme();
  const isDark = scheme === 'dark';
  return {
    isDark,
    colors: isDark ? Colors.dark : Colors.light,
  };
}
Profile > Appearance szekció (ha marad):
Ha az Appearance szekció megmarad a Profile-ban (téma törlés után),
csak ezt tartalmazza:
Appearance
──────────
Display Mode
  ○ Light
  ● Dark       ← ha a rendszer dark
  ○ Auto (System default)   ← ajánlott
Ez egy egyszerű 3-opciós választó legyen,
ami AsyncStorage-ba menti a preferenciát
és felülírja a system default-ot ha a user explicit választ.
